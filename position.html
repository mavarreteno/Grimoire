sticky

relative until you scroll too much and then becomes fixed

relative position

Positions element relative to where it would normally appear by default. This div and its header have a relative position of 30px from the left

fixed

Fixes the elements position on to the view port. This element is fixed on to the bottom-right of the page(these can be useful for nav bars).

absolute

positioned relative to the nearest positioned ancestor instead of relative to the viewport like fixed

overflow

overflow property only works for block elements with a specified height

  • visible: default the overflow is not clipped. the content renders outside the element's box
  • hidden: the overflow is clipped, and the rest of the content will be invisible
  • scroll: the overflow is clipped, and a scrollbar is added to see the rest of the content
  • auto: similar to scroll, but it adds scrollbars only when necessary
